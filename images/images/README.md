# JPP Law — image drop-in guide

Drop correctly-named files into this `images/` folder, commit, and Vercel will redeploy.
Any file you haven't added yet gracefully falls back to the labelled placeholder — so you can add photos one at a time.

Format: JPEG or WebP (WebP preferred for size). Keep each file roughly under 300–400KB.
